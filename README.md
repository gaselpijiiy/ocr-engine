# OCR Engine

## Cara menjalankan

1. Clone repo
git clone https://github.com/gaselpijiiy/ocr-engine.git

2. Masuk folder
cd ocr-engine

3. Buat virtual environment
python -m venv venv

4. Aktifkan venv
venv\Scripts\activate   (Windows)

5. Install dependency
pip install -r requirements.txt

6. Jalankan test
python test_ocr.py
